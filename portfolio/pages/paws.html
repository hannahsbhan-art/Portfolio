🐶 PAWS

AI 기반 반려견 라이프 통합 플랫폼

프로젝트 소개

PAWS는 반려견 보호자들이 산책 메이트를 찾고 다양한 정보를 공유할 수 있도록 개발한 AI 기반 반려견 라이프 통합 플랫폼입니다.

기술 스택

Java Spring Boot MyBatis Oracle JavaScript AJAX AWS EC2 OpenAI API

주요 기능

  • 회원가입 및 로그인
  • 반려견 프로필 관리
  • DBTI 검사
  • GPS 기반 산책 매칭
  • AI 반려견 상담
  • 실시간 채팅
  • 커뮤니티
  • 마켓
  • 관리자 기능

프로젝트 성과

  • 캡스톤디자인 졸업작품 1인 개발
  • OpenAI API 기반 AI 상담 기능 구현
  • GPS 기반 매칭 시스템 구현
  • AWS EC2 서버 배포
  • Oracle Database 설계 및 구축
