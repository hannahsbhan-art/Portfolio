🏋️ 헬스포인터

AI 기반 운동 · 건강 관리 플랫폼

프로젝트 소개

헬스포인터는 사용자의 신체 정보와 운동 기록을 기반으로 맞춤형 운동 및 식단 정보를 제공하는 AI 기반 건강 관리 플랫폼입니다.

기술 스택

Java Spring Boot MyBatis Oracle Python Flask OpenAI API Google Chart JavaScript

주요 기능

  • 회원가입 및 로그인
  • BMI 자동 계산
  • AI 운동 추천
  • AI 식단 추천
  • 운동 기록 관리
  • SNS 커뮤니티
  • 리뷰 감성 분석
  • 건강 데이터 시각화
  • 관리자 기능

담당 역할

  • 관리자 기능 개발
  • 운동 카테고리 관리
  • 추천 기능 구현
  • 로그인 이력 관리
  • OpenAI 기반 AI 기능 개발

프로젝트 성과

  • Spring Boot ↔ Flask 서버 연동
  • OpenAI API 활용 경험 확보
  • Google Chart 기반 데이터 시각화 구현
  • 리뷰 감성 분석 및 키워드 추출 기능 구현
  • Oracle Database 설계 및 구축

개발 환경

  • 개발 기간 : 2024.06 ~ 2024.07
