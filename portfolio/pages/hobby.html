대학생 맞춤형 취미 추천 플랫폼

프로젝트 소개

숨쉴취미는 대학생들이 자신의 성향과 관심사에 맞는 취미를 찾고 다양한 활동 정보를 공유할 수 있도록 개발한 취미 추천 플랫폼입니다.

기술 스택

HTML5 CSS3 JavaScript Java Spring Boot Oracle GitHub

주요 기능

  • 회원가입 및 로그인
  • 취미 카테고리 조회
  • 취미 추천 서비스
  • 커뮤니티 게시판
  • 댓글 및 추천 기능
  • 마이페이지
  • 콘텐츠 조회 기능
  • 관리자 기능

담당 역할

  • 데이터베이스 설계
  • 백엔드 기능 개발
  • 프론트엔드 UI 구현
  • 게시판 기능 구현
  • 프로젝트 테스트 및 유지보수

프로젝트 목표

  • 대학생들의 여가 활동 지원
  • 다양한 취미 정보 제공
  • 사용자 간 커뮤니티 활성화
  • 맞춤형 취미 추천 서비스 제공

배운 점

  • 웹 서비스 개발 전반 경험
  • 데이터베이스 모델링 경험
  • 사용자 중심 UI 설계 경험
  • 팀 프로젝트 협업 능력 향상
  • GitHub 기반 형상관리 경험
